# VTPEH6270-DataProject
# Healthy Minds Study: Technology, AI, and Mental Health

## Description
This project analyzes data from the Healthy Minds Study to explore how technology use and emerging AI engagement relate to mental health outcomes, social interaction, and age differences.

## Author
Hana Bauta  
Cornell Univeristy - MPH Candidate

## Research Question
How does technology and AI use influence mental health and social interaction across different demographic groups?

## Data Source
Healthy Minds Study (HMS) Open Source Dataset

## Project Structure
- 'data/raw/' → original datasets
- 'data/processed/' → cleaned dataset
- 'scripts/' → R scripts for the analysis and simulation
- 'outputs/figures/' → visualizations
- 'outputs/reports/' → reports/results

## Methods
- Data cleaning and exploration
- Statistical analysis of relationships between variables
- Simulation modeling

## How to Run
1. Open RStudio
2. Run scripts in the 'scripts/' folder
3. Outputs will be saved in the 'outputs/' folder

## Outputs
- Visualizations of tech use vs mental health
- Age-based and gender-based comparisons
- Simulation results
